## 🎯 部署步骤（3分钟）

### 步骤 1：访问 Vercel

1. **打开浏览器**，访问：https://vercel.com/new
2. **使用 GitHub 登录**（如果还没登录，点击 "Continue with GitHub"）
3. **授权**：允许 Vercel 访问您的 GitHub 仓库

### 步骤 2：导入项目

1. 在 Vercel 首页，点击 **"Add New..."** → **"Project"**
2. 在仓库列表中，找到并选择 **`bookforMX`** 或 **`DDDDDGCSM/bookforMX`**
3. 点击 **"Import"**

### 步骤 3：配置并部署

Vercel 会自动检测到 Flask 项目，**保持所有默认设置**即可：

- ✅ **Framework Preset**: Other（或自动检测）
- ✅ **Root Directory**: `./`
- ✅ **Build Command**: （留空）
- ✅ **Output Directory**: （留空）
- ✅ **Install Command**: （留空）

**直接点击 "Deploy" 按钮**

### 步骤 4：等待部署（1-2分钟）

- Vercel 会自动：
  - 安装依赖（Flask, Flask-CORS 等）
  - 构建项目
  - 部署到全球 CDN

### 步骤 5：完成！

看到 **"🎉 Congratulations!"** 表示部署成功

点击 **"Visit"** 或复制部署链接，例如：
```
https://bookformx.vercel.app
```

## 🆘 如果遇到问题

### 问题1：找不到仓库

**解决**：
1. 确认 GitHub 仓库是公开的，或
2. 在 Vercel 设置中连接您的 GitHub 账号

### 问题2：部署失败

**解决**：
1. 查看 Vercel 部署日志（点击部署记录）
2. 检查错误信息
3. 确认 `vercel.json` 和 `requirements.txt` 存在

### 问题3：页面显示 404

**解决**：
1. 确认 `vercel.json` 配置正确
2. 检查路由配置
3. 查看 Vercel 日志
