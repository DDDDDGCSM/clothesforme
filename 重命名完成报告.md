## 📝 技术细节

### 重命名规则
- 使用连字符（-）分隔单词
- 使用小写字母
- 保持文件扩展名不变
- 确保文件名唯一且有意义

### 路径映射
所有路径从：
```
/static/墨西哥书/中文文件名.扩展名
```
改为：
```
/static/墨西哥书/英文文件名.扩展名
```
