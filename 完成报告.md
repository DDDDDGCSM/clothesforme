## 🚀 部署步骤

### 在 GitHub 创建新仓库

1. **访问 GitHub**：
   - 打开 https://github.com/new
   - 或使用 GitHub CLI: `gh repo create clothesforme --public`

2. **仓库设置**：
   - 仓库名称：`clothesforme` 或 `bookforME`
   - 描述：`Middle East Clothes Exchange Platform`
   - 设置为 Public 或 Private
   - **不要**初始化 README（我们已经有了）

3. **推送代码**：
```bash
cd "/Users/a58/cursor/归档/OK 调研/bookforME"

# 如果还没有初始化 git
git init
git add .
git commit -m "Initial commit: ClothesForME - Middle East clothes exchange platform"

# 添加远程仓库
git remote add origin https://github.com/your-username/clothesforme.git
git branch -M main
git push -u origin main
```

### 部署到 Vercel

1. **访问 Vercel**：
   - 打开 https://vercel.com/new
   - 使用 GitHub 登录

2. **导入项目**：
   - 选择刚创建的 `clothesforme` 仓库
   - Vercel 会自动检测 Flask 项目

3. **部署设置**：
   - Framework Preset: Other
   - Root Directory: `./`
   - Build Command: （留空）
   - Output Directory: （留空）

4. **点击 Deploy**：
   - 等待 1-2 分钟
   - 部署完成后会获得链接，例如：`https://clothesforme.vercel.app`
