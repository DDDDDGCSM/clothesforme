## 🚀 部署说明

### 快速部署

1. **使用一键部署脚本**:
```bash
cd "/Users/a58/cursor/归档/OK 调研/bookforME"
python3 一键部署.py your_github_token
```

2. **使用完全自动部署**:
```bash
python3 完全自动部署.py github_token vercel_token
```

3. **手动部署**:
   - 访问 https://vercel.com/new
   - 导入 GitHub 仓库: `your-username/bookforME`
   - 点击 "Deploy"

### 部署后访问

部署成功后，您将获得一个 Vercel 链接，例如：
```
https://bookforme.vercel.app
```
